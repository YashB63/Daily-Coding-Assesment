Lowest Common Multiple (LCM)

Problem statement

Find the LCM of two given integers. 
Fill in a function that takes as inputs two integers and return their LCM.

Example 1

Input

input1: 2

input2: 3

Output 

6

Explanation

The LCM of 2 and 3 is 6

Example 2

Input

input1: 5

input2: 65

Output

65

Explanation 

The LCM of 5 and 65 is 65.

Input format :
input1: First integer

input2: Second integer

Output format :
Return the LCM of input numbers

Code constraints :
1 <= input1 < 109

1 <= input2 < 109

Sample test cases :
Input 1 :
2
3
Output 1 : 6
Input 2 :
5
65
Output 2 : 65
